### ~~19. Three-way divergence heuristic is weaker than before~~ INVALID

The `local_said.is_some()` check in the `Failed` branch means: local has state, remote had events, and merge failed. This is a reasonable heuristic — if local and remote agreed, the batch fetch would return no events (no merge needed). Recording a seen SAID for this case only prevents retrying the same remote SAID; a new SAID from a new event resumes retries. Resolving category — acceptable trade-off.
